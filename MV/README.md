# MangaVerse - Сайт для чтения манги

Современный минималистичный сайт для чтения манги с PHP/SQL backend и чистым фронтендом.

## 🚀 Особенности

- **Упрощенный дизайн** - убраны избыточные анимации и AI-подобные эффекты
- **Единый стиль** - консистентный дизайн на всех страницах
- **Inline-авторизация** - без всплывающих окон, с улучшенным UX
- **Умный поиск** - с автодополнением и подсказками
- **PHP/SQL Backend** - полноценная база данных для пользователей и манги
- **Адаптивный интерфейс** - темная/светлая тема

## 📋 Требования

- PHP 7.4+
- MySQL/MariaDB
- Веб-сервер (Apache/Nginx)

## 🛠️ Установка

### 1. Настройка базы данных

Создайте базу данных и импортируйте схему:

```bash
# Через setup.php (рекомендуется)
Откройте в браузере: http://your-domain/setup.php

# Или вручную
mysql -u root -p mangaverse_db < database/schema.sql
```

### 2. Конфигурация

Отредактируйте `api/config.php`:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'mangaverse_db');
define('DB_USER', 'root');
define('DB_PASS', ''); // Ваш пароль
```

### 3. Разместите на сервере

Загрузите все файлы в корневую директорию вашего веб-сервера.

## 📊 База данных

### Основные таблицы:
- `users` - пользователи
- `manga` - манга
- `genres` - жанры
- `chapters` - главы
- `reading_history` - история чтения
- `user_favorites` - избранное
- `user_ratings` - рейтинги пользователей

## 🔗 API Endpoints

### Авторизация:
- `POST /api/auth.php?action=register` - Регистрация
- `POST /api/auth.php?action=login` - Вход
- `POST /api/auth.php?action=logout` - Выход
- `GET /api/auth.php?action=check` - Проверка авторизации

### Манга:
- `GET /api/manga.php?action=search&q=запрос` - Поиск
- `GET /api/manga.php?action=get&slug=название` - Детали манги
- `GET /api/manga.php?action=list` - Список манги
- `GET /api/manga.php?action=genres` - Жанры
- `GET /api/manga.php?action=popular` - Популярное
- `GET /api/manga.php?action=new` - Новинки
- `POST /api/manga.php?action=rate` - Оценить мангу
- `POST /api/manga.php?action=favorite` - Добавить в избранное
